# Aset Gambar Home Page

**Cover / hero home:** taruh gambar background di folder ini, lalu atur URL di **Admin → Home → URL Gambar Hero**.

- **Untuk background hero (cover home):**  
  Simpan file dengan nama misalnya `hero-cover.png` atau `hero.jpg` di folder `public/images/` ini.  
  Lalu di Admin → Home isi **URL Gambar Hero** dengan:  
  **`/images/hero-cover.png`** (sesuaikan dengan nama file Anda).

Contoh nama file dan kegunaan lain (opsional):

| Nama file | Digunakan untuk |
|-----------|-----------------|
| `section-title-bg.png` | **Background section "Meningkatkan Kualitas Hidup"** (putih + motif floral) |
| `hero-cover.png` / `hero.jpg` | **Background cover home** (gambar lansia + overlay gelap) |
| `mengapa-harus-atedia.png` | Section "Mengapa Harus Atedia?" |
| `temukan-kenyamanan.png` | Banner "Temukan Kenyamanan Sejati" |
| `keistimewaan-aman.png` | Kartu Keistimewaan: Aman |
| `keistimewaan-asri.png` | Kartu Keistimewaan: Asri |
| `keistimewaan-akrab.png` | Kartu Keistimewaan: Akrab |

Semua URL gambar bisa diatur lewat **Admin → Home** (teks pembuka & keistimewaan).
